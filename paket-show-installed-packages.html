Paket

paket show-installed-packages

EXPERIMENTAL: Shows all installed top-level packages.

1: 

$ paket show-installed-packages [--all] [project ] [--silent|-s]

Options:

--all: Shows all installed packages (incl. transitive dependencies).

project <string>: Show only packages that are installed in the given project.

--silent [-s]: Doesn't trace other output than installed packages

Sample

Running the command:

1: 

paket show-installed-packages --all -s

will result in:
