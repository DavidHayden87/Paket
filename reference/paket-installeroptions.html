InstallerOptions

Force - Force the download and reinstallation of all packages Hard - Replace package references within project files even if they are not yet adhering to the Paket's conventions (and hence considered manually managed) Redirects - Create binding redirects for the NuGet packages OnlyReferenced - Only install packages that are referenced in paket.references files.
